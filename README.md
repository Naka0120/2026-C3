# 2026年度 $C^{3}$
**〜ビル風を用いた環境音響生成の計算論的デザイン環境（パイプオルガン・デジタルツイン）〜**

## 1. 概要 (Overview)
本プロジェクトは、都市部に発生する局所的な強風（ビル風）などの流体エネルギーを利用し、環境音響を意図的にデザインするためのシミュレーション環境です。
格子ボルツマン法（LBM）による流体解析と、Web Audio API・Three.jsを用いたフル3Dの「計算論的デザイン（Computational Design）」の手法を用いて、リアルタイムに風の流れとパイプオルガンの音色を可視化・可聴化します。

## 2. アーキテクチャ (Architecture)
計算負荷の高い流体シミュレーションと、UI/音響・3D描画を分離したクライアント・サーバーモデルを採用しています。

* **Backend (Python):** 
    * `NumPy` と `Numba` (JITコンパイル) を用いた高速なLBM流体計算エンジン。
    * 物理法則に基づくパイプの開管・閉管共鳴、開口端補正、ストローハル効果（エッジトーン）、および倍音（オーバーブロウ）の計算。
    * `FastAPI` と `WebSockets` を用いたシミュレーション結果（速度場・共鳴状態）のリアルタイム配信サーバー。
* **Frontend (Web/Browser):**
    * `Three.js` によるフル3Dビューア。気流のパーティクルを風速に応じたヒートマップ色（青〜赤）でリアルタイムレンダリング。
    * `Web Audio API` を用いた動的な環境音の合成。対数カーブを用いた音量制御や、クリッピング防止機能などを実装。

## 4. プロジェクトの進捗・実装済み機能 (Features)

### フル3Dビジュアライゼーション
2Dヒートマップの描画から、Three.jsを用いたフル3Dビューアへと進化しました。気流の速度場を無数のパーティクルで表現し、風速に応じて色（青→緑→黄→赤）が変化する物理的で美しい表現を実現しました。

### パイプオルガン音響モデルの実装（エッジトーンと気柱共鳴）
初期フェーズで想定していた「単一の障害物によるカルマン渦（エオルス音）」から、より複雑な楽器の物理モデルへと進化しました。現在は、風がパイプの開口部（エッジ）に当たって発生する**エッジトーン（Edge tone）**と、パイプ内部の空気がバネのように振動する**気柱共鳴**の相互作用によって音が鳴る仕組みを構築しています。
- **ストローハル数の数式の応用**: エッジトーンによる空気の振動周波数は、カルマン渦の発生周波数を求める「ストローハル数の公式（$f = St \times V / D$）」と同じ数式モデルを用いて計算できるという流体力学の特性を利用しています。これにより、既存の数式を活かしたまま、カルマン渦シミュレータをパイプオルガンシミュレータへと昇華させています。
- **開口端補正**: パイプの幅（直径）に基づく物理的な端面補正係数を導入し、現実の管楽器に忠実なピッチチューニングを実現。
- **オーバーブロウ（倍音）**: 風速が一定の閾値（約12〜13m/s以上）を超えた際、基音から第3倍音へとピッチが跳躍する現象（ヒステリシスを伴う状態遷移）を再現。
- **※スケール補正について（デフォルメ）**: 現実のフルートやリコーダーの吹き口は数mm程度ですが、本シミュレータでは流体の渦を視覚的に観察しやすくするため、パイプ幅を10cm（100mm）などあえて巨大に設定しています。これに伴い、現実と同じ周波数（C4等）でエッジトーンを発生させようとすると「台風以上の非現実的な風速」が必要になるため、音響計算の内部では**風速に「26倍」のゲイン補正**をかけ、そよ風（4m/s）や強風（13m/s）といった日常的な風速で共鳴やオーバーブロウが観察できるように意図的な簡単化（ゲーム的補正）を行っています。

### 完璧な和音モード（Perfect Chord）
LBMの物理グリッド（整数）による音程のズレを解消するため、シミュレーション空間の解像度とは独立して「音響計算用の浮動小数点座標」を保持するアーキテクチャを採用しました。これにより、純正律・平均律に極めて近い正確な「ド・ミ・ソ」の和音（Cメジャーなど）を奏でることが可能になりました。

## 5. 環境構築と実行方法 (Setup & Usage)

### 5.1 依存パッケージのインストール
Pythonの仮想環境を作成し、必要なライブラリをインストールします。

```bash
python -m venv C3

# Windowsの場合:
.\C3\Scripts\activate
# Mac/Linuxの場合:
source C3/bin/activate

# 依存パッケージのインストール
pip install -r requirements.txt
pip install fastapi uvicorn websockets numba numpy
```

### 5.2 サーバーの起動
プロジェクトルートで以下のコマンドを実行し、シミュレーションエンジンを起動します。

```bash
python app.py
```

### 5.3 シミュレータへのアクセス
Webブラウザ（Chrome、Edge、Safari等）を開き、以下のURLにアクセスしてください。
`http://localhost:8000`

画面上のスライダーを操作することで、リアルタイムにシミュレーションへ干渉できます。

* **風速 (m/s)**: 流体シミュレーションのベースとなる風速（大元の風の強さ）を設定します。
* **パイプ幅・パイプ深さ (単位: mm)**: シミュレータ内では1グリッド=1cm（10mm）として計算されますが、UI上のスライダーは直感的に設定できるよう**「ミリメートル（mm）」表記**となっています。（例: 幅100 = 100mm = 10cm）

スライダーを動かすと3Dビューア上で気流のパーティクルが変化し、風速とパイプ寸法から算出された共鳴状態に応じた音が鳴ります。また、「Perfect C-Chord」ボタンを押すことで、チューニングされた理想的な和音設定を簡単に体験できます。
