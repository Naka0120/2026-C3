# 2026年度 $C^{3}$
**〜ビル風を用いた環境音響生成の計算論的デザイン環境〜**


## 1. 概要 (Overview)
本プロジェクトは、都市部に発生する局所的な強風（ビル風）と、障害物によって発生する「カルマン渦」を利用し、環境音響（流体音）を意図的にデザインするためのシミュレーション環境です。
物理的な工作を行わず、格子ボルツマン法（LBM）による流体解析とWeb Audio APIを組み合わせた「計算論的デザイン（Computational Design）」の手法を用いて、リアルタイムに風の流れと音を可視化・可聴化します。

## 2. アーキテクチャ (Architecture)
計算負荷の高い流体シミュレーションと、UI/音響生成を分離したクライアント・サーバーモデルを採用しています。

* **Backend (Python):** * `NumPy` と `Numba` (JITコンパイル) を用いた高速なLBM流体計算エンジン。
    * `FastAPI` と `WebSockets` を用いたシミュレーション結果のリアルタイム配信サーバー。
* **Frontend (Web/Browser):**
    * HTML/Vanilla JSによる軽量なUI。
    * Pythonから受信したBase64画像（ヒートマップ）のリアルタイム描画。
    * ストローハル効果の計算結果に基づく、`Web Audio API` を用いた動的な環境音の合成。

## 3. ディレクトリ構成 (Directory Structure)
computational_wind_chime/
 ├── config.py         # シミュレーションの解像度や物理パラメータ(Re数など)の一元管理

 ├── app.py            # FastAPIベースのWebサーバー（WebSocket通信・画像エンコード）
 ├── index.html        # ブラウザ表示用のフロントエンド（UI・音響合成）
 ├── requirements.txt  # Python依存パッケージリスト
 └── README.md         # 本ドキュメント

## 4. プロジェクト・ロードマップ (Phases)

### Phase 1: ストローハル効果の関数化（完了）
流体力学におけるカルマン渦の発生周期（ストローハル効果）の公式をベースに、入力値（局所風速 V、円柱の直径 D）から出力値（基本周波数 f）を算出する軽量な関数（acoustics.py）を実装。

### Phase 2: 高速ビル風シミュレータの構築（完了）
格子ボルツマン法（LBM）を採用し、2Dグリッド空間上における気流の振る舞いを計算するプログラムを実装。NumPyとNumbaによる高速化を行い、FastAPIを通じてブラウザへリアルタイムに計算結果（画像と周波数データ）を配信するパイプラインを構築。

### Phase 3: 逆問題アプローチによる自動配置アルゴリズムの実装（次フェーズ）
設定した「目標とする和音（例: Cmaj7）」から、構造物の「最適な直径と配置座標」を逆算・探索するアルゴリズムを実装する。遺伝的アルゴリズム（GA）や勾配法を用いてLBMシミュレータを反復実行し、最適な配置を自動生成する。

### Phase 4: 数値データの可聴化と結果検証（最終フェーズ）
最適化された設計データから空間全体の音響分布を算出し、複数の正弦波を合成して実際の環境音として出力・評価する。

## 5. 環境構築と実行方法 (Setup & Usage)

### 5.1 依存パッケージのインストール
Pythonの仮想環境を作成し、必要なライブラリをインストールします。

python -m venv venv

# Windowsの場合:
.\venv\Scripts\activate
# Mac/Linuxの場合:
source venv/bin/activate

# 依存パッケージのインストール
pip install -r requirements.txt
pip install fastapi uvicorn websockets opencv-python

### 5.2 サーバーの起動
プロジェクトルートで以下のコマンドを実行し、シミュレーションエンジンを起動します。

python app.py

### 5.3 シミュレータへのアクセス
Webブラウザ（Chrome、Edge、Safari等）を開き、以下のURLにアクセスしてください。
http://localhost:8000

画面上のスライダーで「風速」や「柱の太さ」を動かすと、リアルタイムにカルマン渦の挙動が変化します。「音を鳴らす」ボタンを有効にすると、計算された周波数に基づく合成音が再生されます。
